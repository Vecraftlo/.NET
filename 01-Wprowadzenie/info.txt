---- Narzedzie .NET w terminalu

Komendy:
dotnet info -> info o wersjach .net
dotnet new -> lista wszystkich projektow do utworzenia
dotnet new sln -> utworzenie solucji (nazwa jak folder w ktorym jestes)
dotnet new console -> utworzenie nowej aplikacji konsolowej (nazwa jak folder w ktorym jestes)
dotnet sln add xxx -> dodanie folderu z aplikacja do solucji, gdzie xxx to nazwa folderu
dotnet buil -> buildowanie aplikacji w celu zobaczenia warnings/errors (musisz byc w folderze z apka)
dotnet run -> uruchomienie aplikacji (musisz byc w folderze z apka)

mkdir xxx -> tworzy folder o podanej nazwie xxx
rmdir xxx -> usuwa folder o podanej nazwie xxx
cd xxx -> przechodzi kontekst konsoli do danego folderu
cd .. -> wyjsc z folderu o jeden poziom
dir -> pokazuje liste wszystkich plikow i folderow w danej sciezce
cls -> czysci konsole


--- JAK ROBIC FOLDERY Z ZADANIAMI
1. Wchodzi do folderu z zadaniami
2. Tworzysz folder o nazwie jak pdf np: mkdir TypyDanych1
3. Przechodzisz do nowego folderu za pomoca cd, np: cd TypyDanych1
4. Tworzysz aplikacje konsolowa: dotnet new console
5. Jak chcesz odpalic aplikacje to piszesz (musisz byc w srodku folderu z aplikacja konsolowa): dotnet run
